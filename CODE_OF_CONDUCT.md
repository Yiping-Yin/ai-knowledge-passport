# Code of Conduct

本项目希望成为一个对研究者、开发者、写作者和长期学习者都安全、有效、可协作的开源空间。

## Expected Behavior

参与讨论与协作时，请：

- 尊重不同背景、经验和表达方式
- 以问题和证据为中心，而不是做人身攻击
- 对产品、代码和文档给出具体、可执行的反馈
- 在争议场景下优先澄清事实、约束和目标

## Unacceptable Behavior

以下行为不被接受：

- 侮辱、羞辱、威胁或骚扰他人
- 歧视性言论
- 恶意刷屏、挑衅或破坏讨论秩序
- 在未征得同意的情况下公开他人隐私信息
- 明知会伤害项目协作环境的行为

## Enforcement

维护者可对不当行为采取以下措施：

- 要求修正表达或停止相关行为
- 锁定讨论
- 关闭 issue / PR
- 临时或永久限制参与

## Reporting

如果你需要报告行为问题，请不要在公开线程里升级冲突。优先通过私下渠道联系维护者，或在合适时请求维护者介入。

## Project Standard

这个准则的目标不是制造话术正确性，而是保证讨论质量、协作效率和参与安全。
